# 尚可的组件

本项目用于存放一些自己写的组件

- Echarts资金流可视化组件
- 穿梭拖拽排序组件

### 项目运行

```sh
# Project Setup
npm install

# Compile and Hot-Reload for Development
npm run dev

# Type-Check, Compile and Minify for Production
npm run build

# Run Unit Tests with Vitest
npm run test:unit

# Lint with ESLint
npm run lint
```
